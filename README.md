# RFID Study Notes

这个仓库用于整理和备份我的 RFID 学习资料、阅读笔记与相关文档。

## 内容结构

```text
RFID/
├── 入门资料1/    # RFID 基础资料与参考书
├── 入门资料2/    # C1G2、论文阅读、翻译与学习记录
└── 入门资料3/    # 通信原理相关学习笔记
```

## 主要内容

- RFID 基础知识资料
- EPC Gen2 / C1G2 协议学习笔记
- RFID 相关论文阅读笔记
- 英文资料翻译与整理
- 通信原理补充学习笔记

## 说明

部分超大文件或本地生成文件已通过 `.gitignore` 排除，以避免超过 GitHub 文件大小限制或上传无关数据。

## 使用方式

可以直接作为 Obsidian vault 的一部分打开，也可以在 GitHub 上浏览 Markdown 笔记。
